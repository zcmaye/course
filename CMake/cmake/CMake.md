## CMake安装

### windows

#### 二进制安装

+ 进入[CMake官网](https://cmake.org/download/)

![image-20251112161033058](./assets/image-20251112161033058.png)

+ 下载安装包

![image-20251112161202943](./assets/image-20251112161202943.png)

+ 双击安装

![image-20251112161520577](./assets/image-20251112161520577.png)

+ 安装一直下一步即可，可以选择一下安装位置。

![image-20251112161608424](./assets/image-20251112161608424.png)

+ 安装完成点击finish即可！

![image-20251112161713899](./assets/image-20251112161713899.png)

+ 最后打开终端，输入`cmake --version`，能输出cmake版本则表示cmake安装成功！

  ```bash
  cmake --version
  ```

  输出：

  ```bash
  cmake version 4.2.0-rc3
  
  CMake suite maintained and supported by Kitware (kitware.com/cmake).
  ```

#### MinGW

要想编译代码，还需要一个编译器，如果安装了Vs可以直接使用，没有就需要安装MinGw工具集！
