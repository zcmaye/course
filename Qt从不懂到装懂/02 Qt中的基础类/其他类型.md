## 经时计时器

使用QElapsedTimer。

```cpp
// 推荐使用的API函数
// QElapsedTimer 类
void QElapsedTimer::start();
qint64 QElapsedTimer::restart();
qint64 QElapsedTimer::elapsed() const;
```

+ 主要的使用方法就是测量一个操作耗时多久，例子如下：

  ```cpp
  QElapsedTimer elapse;
  elapse.start();
  
  for(int i = 0;i<10000000;i++);
  
  qDebug()<<elapse.elapsed()<<endl;
  ```
