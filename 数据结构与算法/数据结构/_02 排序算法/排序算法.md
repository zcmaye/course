# 排序算法

在C语言中，排序算法是基础且重要的算法类型，它包括多种不同的方法来对数组或列表中的元素进行排序。

排序算法根据时间复杂度、实现原理和应用场景，通常分为**基本排序**和**高级排序**两大类。

## 基本排序

这类排序算法实现简单，易于理解，但平均时间复杂度较高，通常为 **O(n²)**，适用于小规模数据排序。

### 冒泡排序（Bubble Sort）

冒泡排序是一种简单的排序算法，通过重复遍历要排序的数列，比较并交换相邻元素的顺序，直到整个数列排序完成。这个算法的名字来源于较小的元素会像气泡一样逐渐“浮”到数列的顶端。

**时间复杂度**：O(n^2) **空间复杂度**：O(1)

```c
void BubbleSort(int *arr, int size) {
   int i, j, tmp;
   for (i = 0; i < size - 1; i++) {
       for (j = 0; j < size - i - 1; j++) {
           if (arr[j] > arr[j + 1]) {
               tmp = arr[j];
               arr[j] = arr[j + 1];
               arr[j + 1] = tmp;
           }
       }
   }
}
```

### 选择排序（Selection Sort）

选择排序通过在未排序序列中找到最小（或最大）元素，将其放到已排序序列的末尾，然后再从剩余未排序元素中继续这个过程，直到所有元素都排序完毕。

**时间复杂度**：O(n^2) **空间复杂度**：O(1)

```c
void SelectionSort(int *arr, int size) {
   int i, j, k, tmp;
   for (i = 0; i < size - 1; i++) {
       k = i;
       for (j = i + 1; j < size; j++) {
           if (arr[j] < arr[k]) {
               k = j;
           }
       }
       tmp = arr[k];
       arr[k] = arr[i];
       arr[i] = tmp;
   }
}
```



### 插入排序（Insertion Sort）

插入排序通过构建有序序列，对未排序数据从后向前扫描，找到相应位置并插入。适用于少量数据的排序，是稳定的排序方法。

**时间复杂度**：O(n^2) **空间复杂度**：O(1)

```c
void InsertionSort(int *arr, int size) {
   int i, j, tmp;
   for (i = 1; i < size; i++) {
       if (arr[i] < arr[i - 1]) {
           tmp = arr[i];
           for (j = i - 1; j >= 0 && arr[j] > tmp; j--) {
               arr[j + 1] = arr[j];
           }
           arr[j + 1] = tmp;
       }
   }
}
```



### 希尔排序（Shell Sort）

希尔排序是插入排序的一种更高效的改进版本，通过将整个待排序的记录序列分割成若干子序列分别进行直接插入排序。

**时间复杂度**：O(n^1.3) **空间复杂度**：O(1)

```c
void ShellSort(int *arr, int size) {
   int i, j, tmp, increment;
   for (increment = size / 2; increment > 0; increment /= 2) {
       for (i = increment; i < size; i++) {
           tmp = arr[i];
           for (j = i - increment; j >= 0 && tmp < arr[j]; j -= increment) {
               arr[j + increment] = arr[j];
           }
           arr[j + increment] = tmp;
       }
   }
}
```

## 高级排序

这类排序算法实现相对复杂，但时间复杂度较低，通常为 **O(n log n)** 或线性时间复杂度，适用于大规模数据排序。

### 快速排序（Quick Sort）

快速排序是一种分治法策略的排序算法，通过选择一个基准元素，将待排序数组分为两个子数组，一个包含小于基准的元素，另一个包含大于基准的元素，然后递归地排序这两个子数组。

**时间复杂度**：O(nlogn) **空间复杂度**：O(logn)

```cpp
void QuickSort(int *arr, int begin, int end) {
   if (begin < end) {
       int pivot = arr[(begin + end) / 2];
       int i = begin, j = end;
       while (i <= j) {
           while (arr[i] < pivot) i++;
           while (arr[j] > pivot) j--;
           if (i <= j) {
               int temp = arr[i];
               arr[i] = arr[j];
               arr[j] = temp;
               i++;
               j--;
           }
       }
       if (begin < j) QuickSort(arr, begin, j);
       if (end > i) QuickSort(arr, i, end);
   }
}
```

### 归并排序（Merge Sort）

归并排序是一种有效的排序算法，采用分治法的一个典型应用。它将已有序的子序列合并，得到完全有序的序列；即先使每个子序列有序，再使子序列段间有序。

**时间复杂度**：O(nlogn) **空间复杂度**：O(n)

```c
void Merge(int *arr, int l, int m, int r) {
   int i, j, k;
   int n1 = m - l + 1;
   int n2 = r - m;
   int L[n1], R[n2];
   for (i = 0; i < n1; i++)
       L[i] = arr[l + i];
   for (j = 0; j < n2; j++)
       R[j] = arr[m + 1 + j];
   i = 0;
   j = 0;
   k = l;
   while (i < n1 && j < n2) {
       if (L[i] <= R[j]) {
           arr[k] = L[i];
           i++;
       } else {
           arr[k] = R[j];
           j++;
       }
       k++;
   }
   while (i < n1) {
       arr[k] = L[i];
       i++;
       k++;
   }
   while (j < n2) {
       arr[k] = R[j];
       j++;
       k++;
   }
}
void MergeSort(int *arr, int l, int r) {
   if (l < r) {
       int m = l + (r - l) / 2;
       MergeSort(arr, l, m);
       MergeSort(arr, m + 1, r);
       Merge(arr, l, m, r);
   }
}
```

### 堆排序（Heap Sort）

**堆排序**是一种基于堆数据结构的排序算法。首先将待排序序列构建成一个大顶堆（或小顶堆），此时堆顶元素即为最大值（或最小值）。然后将堆顶元素与堆末尾元素交换，并减少堆的大小，再重新调整堆结构，重复此过程直到整个序列有序。堆排序利用完全二叉树的性质，通过数组下标即可方便地访问父节点和子节点。

**时间复杂度**：O(n log n) **空间复杂度**：O(1)

```c
// 调整以root为根的子树为最大堆
void heapify(int arr[], int n, int root) {
    int largest = root;
    int left = 2 * root + 1;
    int right = 2 * root + 2;

    if (left < n && arr[left] > arr[largest])
        largest = left;
    if (right < n && arr[right] > arr[largest])
        largest = right;

    if (largest != root) {
        int temp = arr[root];
        arr[root] = arr[largest];
        arr[largest] = temp;
        heapify(arr, n, largest);
    }
}

void heapSort(int arr[], int n) {
    // 构建最大堆
    for (int i = n / 2 - 1; i >= 0; i--)
        heapify(arr, n, i);
    // 依次取出堆顶元素
    for (int i = n - 1; i > 0; i--) {
        int temp = arr[0];
        arr[0] = arr[i];
        arr[i] = temp;
        heapify(arr, i, 0);
    }
}
```

### 计数排序（Counting Sort）

**计数排序**是一种非比较型排序算法。它通过统计每个元素值出现的次数，然后根据元素的大小顺序直接计算出每个元素在排序后数组中的位置。计数排序要求待排序的元素必须是整数，且数据范围（最大值与最小值的差）不能太大，否则会占用过多内存。该算法的核心是使用一个计数数组来记录每个值的出现频次，再通过前缀和确定每个元素的最终位置。

**时间复杂度**：O(n + k) **空间复杂度**：O(k) （k为数据范围大小）

```c
#include <stdlib.h>

void countingSort(int arr[], int n) {
    if (n <= 1) return;

    int max = arr[0], min = arr[0];
    for (int i = 1; i < n; i++) {
        if (arr[i] > max) max = arr[i];
        if (arr[i] < min) min = arr[i];
    }

    int range = max - min + 1;
    int *count = (int *)calloc(range, sizeof(int));

    for (int i = 0; i < n; i++)
        count[arr[i] - min]++;

    for (int i = 1; i < range; i++)
        count[i] += count[i - 1];

    int *output = (int *)malloc(n * sizeof(int));
    for (int i = n - 1; i >= 0; i--) {
        int idx = arr[i] - min;
        output[count[idx] - 1] = arr[i];
        count[idx]--;
    }

    for (int i = 0; i < n; i++)
        arr[i] = output[i];

    free(count);
    free(output);
}
```

### 桶排序（Bucket Sort）

**桶排序**将待排序元素分散到若干个有序的“桶”中，每个桶内的元素再分别进行排序（通常使用插入排序或其他简单排序），最后按桶的顺序依次取出所有元素即可得到有序序列。桶排序适用于数据分布较为均匀的场景，通过合理设置桶的数量和范围，可以将平均时间复杂度降低到线性级别。桶排序的核心在于如何将元素均匀地映射到各个桶中。

**时间复杂度**：平均 O(n + k) **空间复杂度**：O(n + k) （k为桶的数量）

```c
#include <stdlib.h>

void insertionSort(float arr[], int n) {
    for (int i = 1; i < n; i++) {
        float key = arr[i];
        int j = i - 1;
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = key;
    }
}

void bucketSort(float arr[], int n) {
    if (n <= 1) return;

    // 创建n个桶（使用链表存储）
    typedef struct Node {
        float val;
        struct Node *next;
    } Node;

    Node **buckets = (Node **)calloc(n, sizeof(Node *));

    // 将元素分配到桶中
    for (int i = 0; i < n; i++) {
        int idx = (int)(arr[i] * n);
        if (idx >= n) idx = n - 1;

        Node *newNode = (Node *)malloc(sizeof(Node));
        newNode->val = arr[i];
        newNode->next = NULL;

        // 插入排序方式插入桶内
        if (!buckets[idx] || buckets[idx]->val > arr[i]) {
            newNode->next = buckets[idx];
            buckets[idx] = newNode;
        } else {
            Node *curr = buckets[idx];
            while (curr->next && curr->next->val < arr[i])
                curr = curr->next;
            newNode->next = curr->next;
            curr->next = newNode;
        }
    }

    // 收集所有桶中的元素
    int idx = 0;
    for (int i = 0; i < n; i++) {
        Node *curr = buckets[i];
        while (curr) {
            arr[idx++] = curr->val;
            Node *tmp = curr;
            curr = curr->next;
            free(tmp);
        }
    }

    free(buckets);
}
```

### 基数排序（Radix Sort）

**基数排序**是一种非比较型整数排序算法。它按照数字的每一位（从低位到高位或从高位到低位）分别进行稳定的排序，通常使用计数排序作为每一位的排序算法。由于整数可以用有限的位数表示，基数排序可以通过对每一位的排序来达到整体有序的效果。对于十进制整数，需要按照个位、十位、百位...依次排序；每一轮排序后，数字就会按照当前位的大小有序排列，最终所有位排序完成后整个序列即为有序。

**时间复杂度**：O(d × (n + k)) **空间复杂度**：O(n + k) （d为位数，k为进制基数）

```c
#include <stdlib.h>

int getDigit(int num, int pos) {
    for (int i = 0; i < pos; i++)
        num /= 10;
    return num % 10;
}

void countingSortByDigit(int arr[], int n, int pos) {
    const int BASE = 10;
    int count[BASE] = {0};
    int *output = (int *)malloc(n * sizeof(int));

    for (int i = 0; i < n; i++) {
        int digit = getDigit(arr[i], pos);
        count[digit]++;
    }

    for (int i = 1; i < BASE; i++)
        count[i] += count[i - 1];

    for (int i = n - 1; i >= 0; i--) {
        int digit = getDigit(arr[i], pos);
        output[count[digit] - 1] = arr[i];
        count[digit]--;
    }

    for (int i = 0; i < n; i++)
        arr[i] = output[i];

    free(output);
}

void radixSort(int arr[], int n) {
    if (n <= 1) return;

    int max = arr[0];
    for (int i = 1; i < n; i++) {
        if (arr[i] > max) max = arr[i];
    }

    int maxDigits = 0;
    while (max > 0) {
        maxDigits++;
        max /= 10;
    }

    for (int pos = 0; pos < maxDigits; pos++)
        countingSortByDigit(arr, n, pos);
}
```

## 复杂度分类对比表

| 分类         | 算法名称 | 平均时间复杂度 | 最坏时间复杂度 | 空间复杂度 | 稳定性   |
| :----------- | :------- | :------------- | :------------- | :--------- | :------- |
| **基本排序** | 冒泡排序 | O(n²)          | O(n²)          | O(1)       | ✓ 稳定   |
|              | 选择排序 | O(n²)          | O(n²)          | O(1)       | ✗ 不稳定 |
|              | 插入排序 | O(n²)          | O(n²)          | O(1)       | ✓ 稳定   |
|              | 希尔排序 | O(n log² n)    | O(n^(3/2))     | O(1)       | ✗ 不稳定 |
| **高级排序** | 归并排序 | O(n log n)     | O(n log n)     | O(n)       | ✓ 稳定   |
|              | 快速排序 | O(n log n)     | O(n²)          | O(log n)   | ✗ 不稳定 |
|              | 堆排序   | O(n log n)     | O(n log n)     | O(1)       | ✗ 不稳定 |
|              | 计数排序 | O(n + k)       | O(n + k)       | O(k)       | ✓ 稳定   |
|              | 桶排序   | O(n + k)       | O(n²)          | O(n + k)   | ✓ 稳定   |
|              | 基数排序 | O(d(n + k))    | O(d(n + k))    | O(n + k)   | ✓ 稳定   |

> **稳定性**是指：在待排序的序列中，如果存在两个**相等的元素**，排序后它们的**相对前后顺序保持不变**，则该排序算法是稳定的；否则是不稳定的

## 选择建议

| 数据规模          | 数据特点          | 推荐算法                     |
| :---------------- | :---------------- | :--------------------------- |
| 小规模（n < 100） | 任意              | 插入排序（常数小，实现简单） |
| 大规模            | 无特殊要求        | 快速排序（平均最快）         |
| 大规模            | 要求稳定性        | 归并排序                     |
| 大规模            | 内存受限          | 堆排序（原地排序）           |
| 整数              | 范围较小（k ≈ n） | 计数排序                     |
| 浮点数            | 分布均匀          | 桶排序                       |
| 整数/字符串       | 位数较少          | 基数排序                     |
| 基本有序          | 近乎有序          | 插入排序（接近O(n)）         |

**总结**：基本排序代码简单但效率低，适用于小数据量；高级排序效率高但实现复杂，适用于大数据量。实际开发中，C语言的 `qsort()` 函数通常结合了多种排序算法（如内省排序），可根据需求选择合适的算法。
